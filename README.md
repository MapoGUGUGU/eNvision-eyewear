# eNVision – Optical Store Website
A modern, responsive front-end website for an optical store that provides elegant and seamless online shopping experience. This project focuses on creating a clean, premium user interface.

## 📖 Overview
eNVision was developed as a front-end web dev project to make a demo of a responsive web design, user interface design, and interactive web functionality using only HTML, CSS, and JavaScript. <br>

Inspired by modern e-commerce websites, eNVision offers a professional shopping experience where users can browse eyewear collections, explore product details, and interact with various website features.

## ✨ Features
- Modern landing page
- Product catalog
- Product search
-  Category filtering Product detail pages
- Shopping cart interface
- Wishlist interface
- Fully responsive design
- Premium and minimalist UI
- Interactive elements powered by JavaScript
- Form validation for user inputs

## 🛠️ Built With
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- IDE: Visual Studio Code

## 🎨 Design Philosophy
The website was designed with these primary goals:
- Modern: Clean layouts with intuitive navigation.
- Elegant: Minimalistic design inspired by premium eyewear brands.

The color palette emphasizes a luxurious shopping experience through the use of white, brown, and gold accents.
